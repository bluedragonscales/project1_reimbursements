User Stories

Employee Stories

As a(an) I want to So that
Employee log in I can check my reimbursements
Employee submit new reimbursements I can get money back
Employee review my reimbursements I can view their status
Employee log out my reimbursement info won't remain available

Manager Stories

As a(an) I want to So that
Manager log in to view employees reimbursement requests and status
Manager approve/deny reimbursement requests I can mark them legitamate or not
Manager leave messages on requests to tell the employee why they were approved/denied
Manager view past reimbursement requests I can check previous decisions
Manager view reimbursement statistics I can keep track of employee spending
Manager log out my decision process is not viewable

System Stories

As a(an) I want to
part of the system reject failed login attempts
part of the system reject negative values for reimbursements
part of the system reject non-numeric values for reimbursements
